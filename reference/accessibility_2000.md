# Accessibility to Cities in 2000

This resource provides global maps of travel time to cities of 50,000 or
more people in year 2000. Accessibility refers to the ease with which
larger cities can be reached from a certain location. This dataset
represents travel time to major cities globally as of the year 2000,
encoded in minutes. The data is essential for historical analyses, such
as understanding the impact of accessibility on land use and
socio-economic outcomes during this period.

## Usage

``` r
get_accessibility_2000()
```

## Source

https://forobs.jrc.ec.europa.eu/gam

## Value

A function that returns an `sf` footprint object.

## References
